# 📋 Todo-App  

**Todo-App** is a simple task management application that allows users to **add, edit, and delete** their daily tasks.  

## 🚀 Features  
- **Add new tasks**  
- **Edit existing tasks**  
- **Delete tasks**  
- **Mark tasks as completed**  
- **User-friendly interface**  

## 🛠️ Tech Stack

- **Backend**: Django / Flask (Specify your backend)
